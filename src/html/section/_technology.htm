О нас

Основная и дополнительная деятельность

Наша команда создает уникальные продающие сайты, логотипы компаний, дизайн будующих страниц сайта, обучает персоонал заказчика работе с сайтом, составляет броские заголовки страниц и тексты с ключевыми запросами по тематике. Подбирает медиа контент с правами свободного использования или генерирует его. Дополнительно занимаемся парсингом данных, созданием телеграмм ботов, конфигурированием и обучением простых нейросетей.

Технологии которые мы используем

Верстка страниц сайта на чистом HTML5, CSS3, JS. Серверная часть Python, Flask, Django. Административная панель Django. Помогаем в подборе доменного имени и хостинга. Дополнительно настраиваем собственный веб сервер, используем Ubuntu, Nginx, postgresql, Django, Flask.

Гарантии

Работаем по согласованному между сторонами техническому заданию, оплата производиться поэтапно. Готовый сайт отображает одинаково контент на последних трех версиях браузеров (кроссбраузерный), не содержит ошибок HTML верски, CSS стилей, JS кода, при проверке Lighthouse набирает более 90% по всем показателям.
